# Painel Core-ATS em Desenvolvimento
Introdução
Este é um painel em desenvolvimento baseado em TypeScript e javascript que usa pnpm para gerenciar dependências e tarefas de desenvolvimento. Este README fornecerá instruções sobre como configurar o projeto e como responder às perguntas de configuração.

### Pré-requisitos
Node.js e npm instalados (preferencialmente a versão LTS)
pnpm instalado globalmente (npm install -g pnpm)

### Instalação
Siga estas etapas para configurar o projeto:

Clone este repositório para o seu ambiente de desenvolvimento:
```bash
git clone https://github.com/drylian/Core-ATS.git
cd Core-ATS
```

### Instale as dependências usando o pnpm:
```bash
pnpm install
```

### Executando o Painel
Agora que o projeto está configurado e as dependências estão instaladas, você pode iniciar o painel em modo de desenvolvimento. Use o seguinte comando:

#### Configuração
Ao executar o painel pela primeira vez, você será solicitado a fornecer algumas informações de configuração. O painel fará perguntas diretamente no console. Certifique-se de responder com as informações apropriadas. Você pode ajustar essas configurações posteriormente, se necessário, editando o arquivo de configuração ou utilizando comandos específicos do painel.

```bash
pnpm dev
```

Isso iniciará o servidor de desenvolvimento e abrirá o painel no seu navegador padrão. Você pode acessar o painel em http://localhost:3000.

### Personalização
Este painel em desenvolvimento pode ser personalizado de acordo com suas necessidades, se possivel ajude no desenvolvimento se achar/aprender algo que possa ajudar no desenvolvimento.
